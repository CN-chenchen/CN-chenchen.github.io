## JAVA中在OJ上怎么实现多组输入

```java
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
        while(sc.hasNext()) {
    		double a = sc.nextDouble();
    		int a1 = (int)a;
    		int a2 = (int)(a*1000);
    		if(a1* 1000 < a2) {
    			System.out.println("No");
    		}else {
    			System.out.println("Yes");
    		}
        }

	}
}
```
